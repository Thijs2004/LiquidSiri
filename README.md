# LiquidSiri

Make your dumb Siri look smart! A premium liquid glass Siri interface for iOS.

## Features
- **Premium Aesthetics**: Stunning multi-layered waves with customizable colors and depth.

## Compatibility
- Compatible with **iOS 14.0 to 16.x**
- Supports both **Rootless** and **Rootful** jailbreaks.
