# LiquidSiri

Make your dumb Siri look smart! A premium liquid glass Siri interface for iOS.

## Features
- **Dynamic Sizing**: Automatically detects device hardware (iPhone X/11 notches, iPhone 12/13 narrow notches, and wraps around the Dynamic Island on iPhone 14 Pro/15).
- **Responsive Waves**: Real-time microphone audio tracking linked directly to Siri's sound inputs.
- **Premium Aesthetics**: Stunning multi-layered waves with customizable colors and depth.

## Compatibility
- Compatible with **iOS 14.0 to 16.x**
- Supports both **Rootless** and **Rootful** jailbreaks.
